成田空港で飛行機に乗る手順

駅から第1ターミナルへ上がって、北ウイング4Fで便を確認し、そのまま保安検査と搭乗口へ進めば大丈夫です。

STEP 1

駅から第1ターミナルへ

成田空港駅で降りたら、改札を出て第1ターミナルの案内どおりに進みます。

  • 成田空港駅 で降ります。
  • 改札を出たら、右のいちばん奥 に進みます。
  • 第1ターミナル の案内に従って、エスカレーターで 1F に上がります。
STEP 2

北ウイング4Fで便を確認

北ウイング側の4Fへ上がり、出発案内で KE706 を確認します。

  • 北ウイング / North Wing 側の 右側エスカレーター で 4F に上がります。
  • 4Fの出発案内モニターで KE706 と SEOUL / INCHEON を確認します。
STEP 3

保安検査と出国へ進む

オンラインチェックイン済みなら、チェックインカウンターに寄らずそのまま進めます。

  • オンラインチェックイン済みで預け荷物もないので、チェックインカウンターには寄らず、国際線出発 または Security Check の表示に従って保安検査へ進みます。
  • 保安検査の入口で、搭乗券を見せるかスキャンします。
  • リュックとトートバッグをX線検査に通し、必要なら PC、液体類、上着 をトレーに出します。
  • 本人確認の検査が終わったら、そのまま出国審査へ進みます。
  • 出国審査では パスポート と 搭乗券 を出します。
STEP 4

搭乗口へ進んで搭乗

出国後に搭乗口番号を見て、案内どおりに進めば搭乗です。

  • 出国後すぐの案内モニターで、KE706 の 搭乗口番号 を確認します。
  • 搭乗口の表示に従って進み、搭乗口に着いたら搭乗開始まで待ちます。
  • 搭乗時に、もう一度 パスポート と 搭乗券 を出して飛行機に乗ります。
